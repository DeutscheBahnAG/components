# Layout

Sets the basic layout of the whole page.

```js
import { Layout } from '@bahn-x/dbx-web';
```

```jsx static
<Layout variant={Layout.variants.CENTERED}>
  {/* See <Container> documentation */}
</Layout>
```

```jsx static
<Layout variant={Layout.variants.LEFT_ALIGNED}>
  {/* See <Container> documentation */}
</Layout>
```
